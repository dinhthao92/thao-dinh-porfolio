📄 Resume Directory

🚀 Hướng dẫn thêm file PDF Resume

Bước 1: Tạo file PDF từ nội dung resume
  • Sử dụng Word/Google Docs để format nội dung
  • Export thành PDF với tên: thao-dinh-resume.pdf
Bước 2: Thay thế file này
  • Đặt file PDF vào thư mục này: public/resume/
  • Đảm bảo tên file chính xác: thao-dinh-resume.pdf
Bước 3: Deploy lại website
  • Push code lên GitHub
  • Deploy lên Vercel/Netlify
  • Test chức năng download

Lưu ý: File PDF hiện tại chưa tồn tại hoặc bị lỗi. Vui lòng thêm file PDF gốc theo hướng dẫn trên.
